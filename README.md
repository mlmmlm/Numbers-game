# 一款用python编写的儿童智力锻炼游戏
## 说明
在终端输入1到100的整数，根据提示猜测数字大小，
可以有效的锻炼儿童对数字大小的判断能力。
## 使用教程：
- 第一步：
- 安装python环境 [获取地址](https://www.python.org/downloads) 不会请Google。
- 第二步：
- 在终端输入
### windonws: 
```
python game2.py
```
### linux:
```
python3 game2.py
```
